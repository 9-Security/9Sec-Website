Node01 / Governance side / Anonymous task workflow

AD and hybrid identity health check with offline evidence handoff.

Create a one-time task, download the assessment kit, run it inside the customer's AD environment, upload the resulting ZIP, and retrieve the report through a signed report link or task secret. This flow is designed for controlled environments where auto-submit is not acceptable.

Task-scoped Anonymous task ID plus one-time secret. No customer account required for v1.
Hybrid evidence On-prem AD evidence plus optional Entra export packaging in the same assessment ZIP.
Manual upload Evidence stays offline until the customer explicitly returns to the portal and uploads it.
Report delivery Completed tasks expose a signed report portal link and PDF retrieval path.
